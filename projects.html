Projeto VIR - Veículo de Inspeção e Reconhecimento

O Projeto VIR foi uma iniciativa de criar um veículo que fosse útil para atividades dentro de espaços confinados, inspeções e reconhecimento de terrenos.

Com ele é possível intervir e conhecer terrenos por meio terreste ou pelo ar, pois ele é um drone que possui uma esteira, o que possibilita a sua locomoção em diversos tipos de superfícies.

Projeto Soft Stock - Gerenciamento de Estoque

O Projeto Soft Stock foi pensado para gerenciar o estoque de ferramentas de uma empresa no ramo de prestação de serviços, ou seja, manutenções em campo.

Este software é capaz de gerenciar o armazenamento de empréstimos e devoluções de ferramentas e materiais no estoque, tudo porque ele é integrado com uma base de banco de dados muito robusta o MySQL. Fora isso também é possível gerar relatórios para impressão das movimentações de estoque, e visualizar as pendências de devolução (serial do equipamento e requisitante).
